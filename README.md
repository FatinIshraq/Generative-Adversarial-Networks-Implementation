# 🎨 Generative Adversarial Networks (GAN) Implementation

## 📌 Overview
This project implements a Generative Adversarial Network (GAN) from scratch to generate realistic data samples. GANs consist of two neural networks:
- Generator
- Discriminator

These networks compete in an adversarial manner to improve the quality of generated outputs.

---

## 🧠 Concept
GAN training is formulated as a minimax game:
- Generator tries to fool the discriminator
- Discriminator tries to distinguish real vs fake data

---

## ⚙️ Implementation Details
- Framework: TensorFlow
- Custom training loop
- Image visualization using Matplotlib

---

## 📊 Results
The model progressively improves its generated samples during training.
